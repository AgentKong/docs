# Quick Setup

<Steps>
  <Step title="Get your API key">
    Go to **Settings → API Keys → Generate New Key** in your Cortana AI dashboard. Enable all read scopes.
  </Step>
  <Step title="Copy your Business ID">
    Find it in your browser URL: `app.usecortana.ai/business/your-id-here/...`
  </Step>
  <Step title="Add the MCP server">
    Pick your AI tool below and paste the configuration.
  </Step>
</Steps>

## Claude.ai (Remote Connector)

The fastest way — no installation needed. Works directly in your browser at [claude.ai](https://claude.ai).

1. Go to **Claude.ai → Settings → Connectors**
2. Click **"Add custom connector"**
3. Enter name: `Cortana AI`
4. Enter URL: `https://app.usecortana.ai/api/mcp`
5. Click **Add**

Start chatting — Claude will discover all Cortana AI tools automatically.

## Claude Desktop

Open **Claude Desktop → Settings → Developer → Edit Config** and add:

```json claude_desktop_config.json
{
  "mcpServers": {
    "cortana-ai": {
      "command": "npx",
      "args": ["-y", "@cortana-ai/mcp-server@latest"],
      "env": {
        "CORTANA_API_KEY": "sk-ak-your-api-key",
        "CORTANA_BUSINESS_ID": "your-business-id",
        "CORTANA_API_URL": "https://app.usecortana.ai"
      }
    }
  }
}
```

Restart Claude Desktop. You'll see a hammer icon in the chat input — your tools are loaded.

## Claude Code

```bash
claude mcp add cortana-ai -- npx -y @cortana-ai/mcp-server@latest
```

Then set environment variables:

```bash
export CORTANA_API_KEY="sk-ak-your-api-key"
export CORTANA_BUSINESS_ID="your-business-id"
export CORTANA_API_URL="https://app.usecortana.ai"
```

## Configuration Reference

| Variable | Required | Description |
|----------|----------|-------------|
| `CORTANA_API_KEY` | Yes | Your API key starting with `sk-ak-` |
| `CORTANA_BUSINESS_ID` | Yes | Business UUID from your dashboard URL |
| `CORTANA_API_URL` | No | API base URL. Defaults to `https://app.usecortana.ai` |
